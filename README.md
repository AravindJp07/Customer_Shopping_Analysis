# 📊 End-to-End Data Analytics Project

## Overview

This project demonstrates a complete end-to-end data analytics workflow, transforming raw data into meaningful business insights. The project covers data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, interactive dashboard development, report creation, and presentation design using industry-standard tools.

The goal is to showcase practical data analytics skills that are commonly used in real-world business environments.

---

## Dataset

- **Source:** CSV Dataset
- **Type:** Business/Sales Data
- **Contents:** Customer, Product, Sales, Profit, Category, Region, Date, Quantity, and other business-related information.

## Project Workflow

### 1. Data Loading
- Imported the dataset into Python.
- Examined data structure and data types.
- Checked dataset dimensions.

### 2. Exploratory Data Analysis (EDA)
- Analyzed data distributions.
- Identified trends and patterns.
- Checked missing values.
- Detected outliers.
- Created visualizations for better understanding.

### 3. Data Cleaning
- Removed duplicate records.
- Handled missing values.
- Corrected inconsistent data.
- Converted data types.
- Prepared the dataset for analysis.

### 4. SQL Analysis
- Imported the cleaned dataset into SQL databases.
- Wrote analytical SQL queries using:
  - SELECT
  - WHERE
  - GROUP BY
  - HAVING
  - ORDER BY
  - JOINs
  - Common Table Expressions (CTEs)
  - Window Functions
  - Aggregate Functions

- Performed business analysis such as:
  - Sales Performance
  - Profit Analysis
  - Regional Performance
  - Customer Analysis
  - Product Performance
  - Monthly Sales Trends

### 5. Power BI Dashboard
Created an interactive dashboard featuring:
- Executive KPI Cards
- Sales & Profit Trends
- Regional Analysis
- Category Performance
- Product Performance
- Customer Insights
- Interactive Filters & Slicers

### 6. Business Report
Prepared a report summarizing:
- Data analysis
- Key findings
- Business insights
- Recommendations

### 7. Presentation
Created a professional presentation using **Gamma** to communicate project objectives, methodology, insights, and recommendations.

## How to Run

1. Clone this repository.
2. Install the required Python libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

3. Open and run the Python notebooks in the following order:
   - Data Loading
   - Exploratory Data Analysis (EDA)
   - Data Cleaning

4. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.

5. Execute the SQL scripts to perform business analysis.

6. Open the Power BI (.pbix) file to explore the interactive dashboard.

7. Review the report and presentation for project insights and recommendations.
